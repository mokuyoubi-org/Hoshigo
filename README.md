# 星碁 / Hoshigo 🌟

Expo（React Native）で開発されたオンライン囲碁アプリです。

---

## 🎮 星碁 / Hoshigoとは？

星碁 / Hoshigo は、React Native（Expo）で構築された囲碁アプリです。
9路盤をベースに、オンライン対戦・組制度・対局結果表示などの機能を実装しています。

囲碁ロジックは外部ライブラリに依存せず、基本部分を独自実装しています。

---

## ✨ 主な機能

- 9路盤対戦
- オンラインマッチング
- ポイントシステム
- 終局処理
- 結果モーダル表示
- 対局データ管理
