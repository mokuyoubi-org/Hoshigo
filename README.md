## 🌟 星碁 / Hoshigoとは？

星碁 / Hoshigo は、モバイル用の囲碁アプリです。
9路盤をベースに、オンライン対戦などの機能を実装しています。

---

## ✨ 主な機能

- 9路盤オンライン対戦 / 自動マッチング
- 自動地計算、地の表示
- 観戦機能
- ポイントシステム
- ぐみシステム

## 🛠 仕組み

- ⛰️ Expo: アプリ本体
- ⚡️ Supabase: データベース。ユーザ、対局の管理
- 🪽 Fly: GnuGoのデプロイ先
- 🐱 RevenueCat: 課金処理
- ✉️ Resend: メールの送信
